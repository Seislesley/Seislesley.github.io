VNC Viewer On Mac

18 Sep 2019, 14:00

在mac上用VNC Viewer远程登陆服务器图形化界面

##配置用户VNC Server，用gnome桌面显示 ssh username@IP ssh 登陆服务器 vi ~/.vnc/xstartup vi进入配置文件

按照上图修改，完成后保存即可生效。主要包括，将文件头前两行取消注释，文件末加入gnome的设置。

##日常使用步骤 1. 在服务器端启动VNC Server ssh username@IP vncserver 启动VNC Server，启动后会返回端口值 如图，1即为本次建立的连接的端口值

2. 在本地端用VNC Viewer远程连接 输入 IP:端口值 即可成功连接

##服务器端VNCServer操作 vncserver -geometry 1080x720 用geometry选项设置显示区域大小，1080x720可根据个人需求修改 vncserver -list 显示当前账户已打开的vncserver显示端口 vncserver -kill :端口值 杀死该端口值的连接
